## Data Science Web Scrape

## Description

I analyzed ski resort data to explore the relationship between resort size and the price of a daily lift ticket. I collected data by scraping information from skiresort.info and then used Python to clean, structure, and prepare the dataset for analysis.

After preparing the data, I conducted exploratory data analysis and created visualizations and graphs to identify and communicate trends, particularly the correlation between resort size and ticket pricing.

## Realtionship to BCIS Learning Goal

This project demonstrates the use of data and computer science platforms to process structured and unstructured data, construct usable datasets, and generate insights that address a business question.

Using Python, I collected and transformed raw web-scraped data into a structured format suitable for analysis. I then applied data analysis and visualization techniques to identify patterns and relationships, supporting the understanding of pricing behavior in the ski industry.

The project also developed critical thinking, creativity, collaboration, and communication skills through problem formulation, exploratory analysis, and the clear presentation of findings in a visual format.
