## SQL Database App

## Description 
In this project, I developed a Java-based database application to store and manage carpooling data. I designed and implemented multiple relational tables using SQL to track users, ride offers, ride requests, and location data.

The application allows users to search for existing users or create new profiles, post ride requests, and find matching rides within the same time window. This project focused on building a functional system that integrates front-end interaction with back-end database management.

## Relationship to BCIS Learning Goal
This project demonstrates active participation in the full software development lifecycle, including planning, design, testing, and implementation. I independently designed the database schema, developed the application logic in Java, and tested the system to ensure accurate data storage and retrieval.

The project required critical thinking and creativity to structure relational data efficiently and design features, such as matching riders based on time and location. Additionally, it strengthened my ability to build integrated software solutions that connect user interaction with underlying database systems, aligning with core BCIS objectives in software development and system design.
